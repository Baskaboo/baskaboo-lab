## How to Read a Session

1. Start with the session's `README.md` for an overview
2. Read `01-*-conversation.md` – the founding conversation
3. Read the AI reports (`02-*`, `03-*`, `04-*`) – different perspectives
4. End with `05-consensus.md` – the unified AI verdict
